# Changelog

Все значимые изменения в этом проекте задокументированы здесь.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
проект следует [Semantic Versioning](https://semver.org/lang/ru/).

---

## [2.0.0] — 2025

### ✨ Добавлено
- Поддержка **нескольких правил** (неограниченное количество пар «канал + триггер»)
- **8 встроенных звуков**: Динь, Тревога, Поп, Колокол, Блип, Срочно, Мягкий, Ретро
- Загрузка **своего звука** (MP3/WAV/OGG) прямо в настройки плагина
- Индивидуальный **слайдер громкости** (0–100%) для каждого правила
- Опция **учёта регистра** для точного поиска триггера
- **Toast-уведомление** Discord при срабатывании (опционально для каждого правила)
- Включение/выключение **каждого правила по отдельности** тоглом
- **Автосохранение** настроек через `BdApi.Data`
- Панель настроек с **живым предпросмотром** звука

### 🔧 Исправлено
- Надёжный поиск FluxDispatcher — **6 методов fallback**
- Устранены утечки памяти при остановке плагина
- Корректная работа при обновлении Discord

### 🗑 Удалено
- Хардкод канала `881306138686148679` — теперь всё настраивается в UI

## [1.2.0] — 2025

### ✨ Добавлено
- 6 методов поиска FluxDispatcher (fallback-цепочка)
- Поиск через `webpackChunkdiscord_app` как последний fallback

### 🔧 Исправлено
- Ошибка «FluxDispatcher не найден» на некоторых версиях Discord
- Замена `getByKeys("actionLogger")` на более надёжную цепочку методов

## [1.1.0] — 2025

### 🔧 Исправлено
- Ошибка «Dispatcher не найден»
